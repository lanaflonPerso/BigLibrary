# BigLibrary

A system which is used to manage a library.

There are three roles :
 - admin
 - reader
 - librarian
 
Each role has it's own functionality that is secured on back-end side.

# Technologies: 

- Servlets, JSP, JSPF, JSTL
- JDBC, SQL, 3NF, DBCP
- JUnit, LOG4J
- HTML5, CSS3, JQuery, Bootstrap
- Maven, Ant, MailAPI
